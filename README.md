# KIOSQANDROID
KIOS Q aplikasi kasir &amp; manajemen keuangan profesional yang dirancang khusus untuk kebutuhan UMKM dan bisnis retail. Berbasis offline-first, aplikasi tetap berjalan lancar tanpa internet dan dilengkapi sistem backup data otomatis/manual untuk memastikan keamanan data usaha tetap terjaga setiap saat.
